Conceptos 
- React es una libreria de Js
- libreria o biblioteca, conjunto de subprogramas para crear algo
- creamos componentes en react 
- componente - interfaz independiente y reusable
  se dividen en : funcionales y de clase
  - funcionales - poderosos dinamicos y demas - es una funcion de Js que 
  retorna un elemento de react en JSX - nombres deben iniciar en MAYUSCULA,
  puede recibir parametros(props(properties))
  - Que son los props - son simplemente parametros - solo pueden ser enviado de
  padre a hijo - no al reves 

  - de clase - anteriores - es una clase de Js que nos devuelve un JSX 
  Caracteristicas 
  - hereda de React-Component
  - debe tener un metodo render
  - puede recibir valores 

- Antes usabamos componentes de clase para trabajar con los estados de los componentes 
  Pero ahora gracias a los Hooks ya nos permiten agregarle estados a nuestros componentes
  
- Hook es una funcion que nos permite trabajar con estados en componentes funcionales 
- Event Listener tambien llamada Event Handler es una funcion para eventos especificos 

- Que es JSX - extension de react para Js - es similar a html - JSX es OPCIONAL

- Atributos - clases,id y demas - class == className 
for == htmlFor

  style={estiloDelObjeto}
